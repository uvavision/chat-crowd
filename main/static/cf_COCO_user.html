Overview
In this interactive drawing game, you will act as an instructor, who describes an image to a painter for drawing the layout of it via a chat interface. The layout of an image includes objects and their positions. All objects are represented by a set of rectangles (proportional to the size of the corresponding objects) and labels. See Figure 2 for an image and its layout on a canvas. English only for text messages.
Note: Do NOT wait for the responses from the painters. If the drawing is not completed, other instructors will continue to provide more instructions. Your contribution will be evaluated later for bonus.

Steps
  1. Login with your "Contributor ID" and your choice of username. Click START to proceed.
  2. If you do this task for the first time, there will be a simple quiz to prepare you for the task. Once you pass the quiz, you will enter the task.
  3. Please review the chat history (previous instructions and the latest painting) and determine what to do, such as
    • - Prove instructions on how to draw some of the tagged objects with descriptions of the types (e.g., person, dog), size and relative location. (press Enter to submit messages).
    • - Answer the questions from the painter or help correct mistakes in the latest painting if necessary.
    • - Please avoid chitchat or duplicated instructions. Do NOT ask the painter to draw any untagged object.
  4. Click
    next task
    to start next ask or
    submit
    or submit the task.
  5. IMPORTANT After you complete the task, please copy the confirmation code to the crowdflower main task.
