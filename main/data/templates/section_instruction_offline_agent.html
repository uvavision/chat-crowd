• There is a simple quiz for the first-time participants. It is mainly for preparing you for the task. After you login (or pass the quiz), you will see the chat interface (left) and the search tool (right).
  1. Review the chat history in area [1]. As shown in the example, you should be able to fill out the search form [2] with the current criteria (Please note that for certain field such as "neighborhood", you need to select from the auto-complete options).
  2. The search results show in two expandable parts,
    • the list of apartment info [3A] and the total of search results.
    • some additional information [3B] to the search results, i.e., count by features.
  3. You can type your response in [4] and press Enter to submit chat.
  4. Please only end the conversation after confirming with the other user.
  5. IMPORTANT Click [6] and obtain the confirmation code to complete the task via CrowdFlower interface. Otherwise, we cannot reward you.
